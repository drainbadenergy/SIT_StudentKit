# 🎓 SIT Student Kit

**SIT Student Kit** is a student-built web platform designed to help incoming students of **Symbiosis Institute of Technology, Pune (SIT Pune)** easily access all essential information about the college, all put in one place.

## 🚀 Project Overview

Information about SIT Pune is currently scattered across multiple sources — websites, social media pages, and student groups — which makes it difficult for new students to get a clear understanding of the college, its facilities, and its culture before joining.

**SIT Student Kit** solves this by providing a centralized, easy-to-navigate portal that consolidates everything students need to know about SIT.

🔗 **Live Website:** [https://sit-student-kit.netlify.app/](https://sit-student-kit.netlify.app/)

---

## 🧠 Problem Statement

Incoming students at SIT Pune face challenges in finding accurate and consolidated information about the college since details are distributed across multiple platforms.  
This project aims to create a unified portal that provides structured, verified, and easily accessible information to improve the onboarding experience for new students.

---

## 💡 Key Features

- 🏫 Overview of SIT Pune — academics, departments, and facilities  
- 👥 Information about clubs, events, and campus life  
- 🗺️ Quick navigation to important resources and contacts  
- 📱 Responsive and user-friendly design  
- ⚙️ Built with scalability in mind for future intranet integration
